# Бустер методиста

Набор инструментов для L&D-методиста, продакт-методиста и фасилитатора. Всё в одном веб-приложении.

**URL:** https://look85-ops.github.io/NAAssistant/methodist-booster/  
**Деплой:** GitHub Pages (https://look85-ops.github.io/NAAssistant/methodist-booster/)  
**Дизайн-система:** `design/TZ_DESIGN.md`

## Инструменты (6)

| Инструмент | Файл | Что делает |
|---|---|---|
| Страдания инженера Иванова | `ivanov.html` | Симулятор учебного дня: прогони сценарий на стрессоустойчивость |
| Активности | `activities.html` | AI-генератор активностей под тему, цель и формат |
| Карты | `cards.html` | Фасилитационные вопросы, 5 категорий, с инструкцией ведущему |
| Аудит | `audit.html` | Диагностика запроса заказчика (Mager & Pipe), готовый бриф |
| Видение | `vision.html` | Мастер видения продукта: от боли до MVP с критериями |
| AI-инструменты | `prompts.html` | Конструктор промптов и советы по работе с AI |

**Стек:** HTML, CSS (tokens/base/components/pages), JS. AI-функции на GH Pages не работают — Активности используют fallback (статичная база).

## Деплой

### Локально
Открой любой `.html` в браузере.

### GitHub Pages (текущий)
Публикуется как часть `NAAssistant` репозитория. Путь: https://look85-ops.github.io/NAAssistant/methodist-booster/

### Примечание
Исторически рассматривался Netlify для AI-функций, но сейчас деплой только на GitHub Pages.

## Убежище

`creature.html` — тёплая страница-поддержка. Пушистое золотистое существо, которое радуется твоему приходу, следит за мышкой/пальцем, машет лапками и мурлычет от поглаживаний.  
**URL:** `creature.html` (в корне)
