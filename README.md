# Бустер методиста

Набор инструментов для L&D-методиста, продакт-методиста и фасилитатора.

**URL:** https://look85-ops.github.io/NAAssistant/methodist-booster/  
**Деплой:** GitHub Pages  
**Дизайн-система:** `design/TZ_DESIGN.md`

## Инструменты

| Инструмент | Файл | Что делает |
|---|---|---|
| Страдания инженера Иванова | `ivanov.html` | Симулятор учебного дня |
| Активности | `activities.html` | AI-генератор активностей под тему, цель и формат |
| Карты | `cards.html` | Фасилитационные вопросы, 5 категорий |
| Аудит | `audit.html` | Диагностика запроса заказчика (Mager & Pipe) |
| Видение | `vision.html` | Мастер видения продукта: JTBD → MVP |
| Промпты | `prompts.html` | 20+ промптов для L&D и конструктор по РЗКФ |
| AI-инструменты | `ai-tools.html` | Каталог проверенных AI-сервисов |

**Стек:** HTML, CSS (tokens/base/components/pages), JS.
