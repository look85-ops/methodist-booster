Command: /webapp-test

Цель:
- Легкий smoke-check статического портфолио из папки `portfolio` перед деплоем (линки, meta, responsive базовая проверка).

Что делает:
- Проверка структуры: наличие `portfolio/_redirects`, основных html.
- Список внутренних ссылок и проверка их наличия.
- Чек-лист webapp-testing (Anthropics) — ручной прогон: заголовки, viewport meta, шрифты, изображения, простая адаптивность.

Как использовать:
1. Запусти локальный сервер на `portfolio` (например, через Live Server или любой статический сервер).
2. Пройди чек-лист из `docs/skills/ADOPTED_SKILLS.md` → Webapp Testing.
3. Зафиксируй найденные проблемы в `docs/BACKLOG.md`.

Примечание:
- Для полной автоматизации используем Playwright Skill по мере необходимости.
