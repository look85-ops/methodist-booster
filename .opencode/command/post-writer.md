# /post-writer

Анализ, оценка и написание LinkedIn-постов.

## Аргументы

| Аргумент | Обязательный | Описание |
|----------|--------------|----------|
| `action` | Нет | `analyze` (по умолчанию), `write`, `improve` |

## Сценарий

1. Определи действие (analyze/write/improve)
2. Если analyze/improve — запроси текст поста или путь к файлу
3. Если write — запроси тему/контекст поста
4. Проведи анализ или сгенерируй варианты
5. Верни: анализ + рекомендации + 2-3 варианта

## Примеры

```
/post-writer
/post-writer analyze
/post-writer write про карьерный переход в IT
```
