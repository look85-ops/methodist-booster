# /create-spec

Превращает краткое описание в подробное техническое задание (ТЗ) для фронтенд-разработчика.

## Аргументы

| Аргумент | Обязательный | Описание |
|----------|--------------|----------|
| `prompt`   | Да           | Краткое описание желаемого сайта/страницы |

## Сценарий

Вызывает агента `frontend_analyst` для генерации ТЗ.
