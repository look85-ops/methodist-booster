# /init-backlog

Инициализация бэклога с нуля или пересоздание структуры.

## Процесс

1. Проверить существование `docs/BACKLOG.md`
   - Если существует → спросить перезаписать или дополнить
   - Если нет → создать с нуля

2. Создать/обновить `docs/BACKLOG.md`:
   ```
   # Бэклог NM.ASSISTANT

   ## Актуальные задачи (месяц год)

   ## Бэклог (неприоритетные)

   ## Завершённые
   ```

3. Добавить запись в AGENTS.md → Commands

## Когда вызывать

Когда пользователь хочет:
- Начать новый бэклог с нуля
- Очистить и пересобрать структуру
- Первый раз настраивает систему

## Ответ

Сообщить пользователю:
- Что создано
- Где что лежит
- Что дальше делать (`/idea` для добавления фич)
