# /career-architect

Персональный карьерный консультант: резюме, LinkedIn, оценка вакансий, карьерные вопросы.

## Аргументы

| Аргумент | Обязательный | Описание |
|----------|--------------|----------|
| `запрос` | Да           | Что нужно сделать (например: "обнови резюме под вакансию Python Developer" или "оцени моё резюме для отклика на эту вакансию") |

## Сценарий

Агент использует свой бриф из `.opencode/agent/career-architect.md`.

1. Определи тип запроса
2. Прочитай существующие файлы из `career/`, если релевантно
3. Выполни задачу
4. При создании/обновлении файлов соблюдай версионность
5. Верни результат в запрошенном формате

## Команды агента

| Команда | Описание |
|---------|----------|
| `/career update-resume <вакансия>` | Актуализировать резюме под вакансию |
| `/career review-resume <файл>` | Оценить резюме по ATS-чеклисту |
| `/career linkedin <задача>` | Создать/обновить структуру LinkedIn |
| `/career cover-letter <вакансия>` | Сгенерировать cover letter |
| `/career question <вопрос>` | Ответить на карьерный вопрос |
