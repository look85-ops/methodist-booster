Command: /enable-skills

Что делает:
- Коротко фиксирует, какие внешние навыки (skills) мы используем, и где они применяются.
- Дает ссылки на инструкции установки в локальные агенты (по желанию).

Как использовать:
1. Посмотри файл `docs/skills/ADOPTED_SKILLS.md`.
2. Если ты используешь Claude/Cursor локально и хочешь автоподсказки:
   - Superpowers: следуй README (obra/superpowers)
   - MarketingSkills: npx skills add coreyhaines31/marketingskills
   - Playwright Skill: plugin install/standalone (см. README)

Заметка:
- Мы не добавляем внешние файлы в репозиторий, только следуем паттернам.
