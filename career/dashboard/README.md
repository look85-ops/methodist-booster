# Карьерный дашборд

CSV-шаблоны для импорта в Google Sheets.

## Импорт

1. Открой https://sheets.new
2. Файл → Импорт → Загрузить → выбери csv-файл
3. В диалоге: «Создать новую таблицу», разделитель — запятая
4. Повтори для второго файла (или добавь лист: + → Импортировать лист)

## Статусы воронки

discovered → evaluating → applied → screening → interview → offer → rejected → accepted
