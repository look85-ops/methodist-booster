# Geometric Pukuru — конструктор персонажа

Каждый Pukuru собирается из 5-7 базовых фигур. Это гарантирует консистентность и возможность собрать сцену руками.

## Базовая анатомия (модуль)

```
     ██████          — голова (круг или скруглённый квадрат)
     ██░░██          — глаза (два маленьких круга)
        ██           — нос (маленький треугольник / отсутствует)

    ██████████       — корпус (прямоугольник со скруглениями)

   ██        ██      — плечи (два круга-сустава)
   ██        ██      — руки (длинные прямоугольники)
   ██        ██      — кисти (круги / маленькие прямоугольники)

   ██        ██      — ноги (прямоугольники)
   ██        ██      — стопы (горизонтальные прямоугольники)
```

## Принципы

1. **Толщина линий:** одинаковая везде — 2-3% от ширины фигуры
2. **Зазоры:** между частями тела всегда есть просвет (1-2%)
3. **Углы:** либо прямые, либо скруглённые (без "случайных" кривых)
4. **Глаза:** всегда два одинаковых круга
5. **Конечности:** всегда пара (две руки, две ноги)

## Вариации (как менять персонажа)

| Часть | Варианты |
|-------|----------|
| Голова | круг, овал, квадрат скруглённый, треугольник с округлыми углами |
| Корпус | прямоугольник, трапеция (расширяется кверху/книзу) |
| Руки | длинные/короткие, с суставами/без, заканчиваются кистью-кругом/прямоугольником |
| Ноги | прямые, слегка согнутые, с большими стопами-платформами |
| Уши/хвост/аксессуары | треугольники, маленькие круги, дополнительные прямоугольники |

## Цветовая система

- Каждый Pukuru = ровно 3 цвета (тело + деталь1 + деталь2)
- Сцена = максимум 6 цветов (3 + 3 на второго персонажа)
- Без полутонов, без теней, без градиентов
