# РЖД 2288р (14.11.2016) — навигация

Источник: распоряжение ОАО «РЖД» №2288р от 14.11.2016 (Инструкция по текущему содержанию железнодорожного пути) и последующие редакции.

Зачем нам это: нормативная база для модулей по текущему содержанию пути. В курс включаем выдержки и применимые пункты.

Где читать: по внутренней подписке (КонсультантПлюс/Гарант) или на корпоративном портале.

Замечание: полный текст не храним в репозитории. При подготовке материалов ссылаться на конкретные пункты/таблицы с датой редакции.
