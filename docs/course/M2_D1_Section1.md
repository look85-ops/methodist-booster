## Эталонные формулы и обозначения

Единицы: мм, м, ‰; округление: до целых мм (правило ≥0.5 → вверх). Допуски при проверке ответов: ±1 мм.

1) Зазоры колёсной пары на прямом участке:

```
δmin = Smin − qmax
δmax = Smax − qmin
```

2) Оптимальная ширина колеи (свободное вписывание):

```
Sопт = qmax + fн − fв + 4 − ∑η
Условие: Sопт ≤ Smax (для данного R по ПТЭ)
```

3) Минимально допустимая ширина (заклинённое вписывание трёхосной тележки):

```
Sзак = qmax + fн + fв + 4 − ∑η ≤ Smax
```

4) Стрелы изгиба рельсовых нитей в кривой:

```
fн = (λ + bн)^2 / (2R)
fв = (λ − bв)^2 / (2R)

где:
  λ = L0/2 — расстояние от центра вращения экипажа до геометрической оси первой колесной пары
  bн — расстояние от геометрической оси первой колесной пары до точки касания гребня с наружной нитью
  bв — расстояние от геометрической оси второй колесной пары до точки касания гребня с внутренней нитью
```

Примечание: использовать паспортные значения qmin/qmax, ∑η, λ, bн, bв; R — по условию; Smax — по ПТЭ/табл. норм для заданного радиуса.

## Ключевые определения (кратко)
- Рельсовая колея — расстояние между внутренними рабочими гранями головок рельсов, измеренное на 13 мм ниже поверхности катания.
- Основные параметры устройства пути: ширина колеи, уровень рельсовых нитей.
- Подуклонка рельсов — наклон головки внутрь колеи для лучшего сопряжения с профилем колеса.

## Быстрые проверки (с эталонами)

БП‑1 (5 мин, индивидуально):
- Дано: Smin = 1516, Smax = 1526; qmin = 1487, qmax = 1509 (локомотив ≤140 км/ч).
- Найти: δmin, δmax; сделать вывод о допустимости.
- Эталон: δmin = 1516−1509 = 7 мм; δmax = 1526−1487 = 39 мм. Вывод: диапазон значительный; контролировать влияние на динамику (виляние/удар гребней).

БП‑2 (5 мин, пары):
- Дано: qmax = 1509; λ = 2100 мм; bн = 4 мм; bв = 4 мм; R = 920 м; ∑η = 1.5 мм; Smax(R=920 м) = 1520 мм.
- Найти: fн, fв, Sопт и сравнить с Smax.
- Эталон: fн ≈ 2 мм; fв ≈ 2 мм; Sопт = 1509 + 2 − 2 + 4 − 1.5 = 1511.5 мм ≤ 1520 мм. Вывод: свободное вписывание обеспечивается при заданных bн/bв; уширение по ПТЭ не требуется.

Примечание: эталон БП‑2 проверяет внимательность к знакам (−fв) и к ∑η.

## Практика 1: расчёт Sопт для ТЭ3 (10+5 мин)

Условие:
- Локомотив ТЭ3: qmax = 1509 мм; L0 = 4200 мм ⇒ λ = 2100 мм; bн = 7 мм; bв = 0 мм (если не задано — принять 0 для оценки чувствительности); ∑η = 1.5 мм; R = 920 м; Smax(920 м) = 1520 мм.

Шаги:
1) fн = (2100 + 7)^2 / (2·920000) ≈ 2.4 → 2 мм
2) fв = (2100 − 0)^2 / (2·920000) ≈ 2.4 → 2 мм
3) Sопт = 1509 + 2 − 2 + 4 − 1.5 = 1511.5 мм
4) Сравнение: 1511.5 ≤ 1520 → свободное вписывание допустимо.

Вывод: уширение по ПТЭ не требуется; принять норму по ПТЭ в зависимости от R.

Замечание по ловушке: часто ошибочно складывают fн и fв или теряют знак «−fв», что завышает Sопт.

## Практика 2: расчёт Sзак для ТЭ3 (10+5 мин)

Условие:
- ТЭ3: qmax = 1509 мм; λ = 2100 мм; bн = 4 мм; bв = 4 мм; ∑η = 15.5 мм (суммарно по трём осям); R = 920 м; Smax(920 м) = 1520 мм.

Шаги:
1) fн = (2100 + 4)^2 / (2·920000) ≈ 2.4 → 2 мм
2) fв = (2100 − 4)^2 / (2·920000) ≈ 2.4 → 2 мм
3) Sзак = 1509 + 2 + 2 + 4 − 15.5 = 1501.5 → 1502 мм (по правилу округления)

Вывод: Sзак ≤ Smax; минимально допустимая ширина ~1502 мм. В эксплуатации принимается ширина по ПТЭ в зависимости от радиуса (для 920 м — 1520 мм).

Замечание по ловушке: ∑η использовать суммарно; не путать с разбегами по одной оси.
