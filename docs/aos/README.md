# AOS артефакты и сборка отчётов

Исходники
- Отчёты и анализ храним в Markdown в `docs/aos/` и `docs/aos/reports/`.
- HTML/PDF считаем артефактами сборки — не коммитим в репозиторий.

Сборка HTML/PDF
- Для HTML используем статический генератор/рендер (
  pandoc или md-to-html) локально.
- Для PDF используем pandoc или wkhtmltopdf локально.

Артефакты
- Сохраняем сгенерированные HTML/PDF в `dist/aos/` (локально), добавляем `dist/` в .gitignore.

Установка Pandoc
- Скачайте с официального сайта: https://pandoc.org/installing.html
- Устанавливать бинарники в репозиторий нельзя (раздувает repo и мешает ревью).
