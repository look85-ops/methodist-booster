# SOP: Локальная разметка комментариев AOS (без внешних сервисов)

Цель: быстро классифицировать комментарии AOS по темам и тонам без внешних сервисов, сохранив приватность данных.

Вход: CSV/TSV выгрузки AOS, колонка с текстом комментария.

Шаги:
1) Подготовка
- Собрать файлы в одну папку `data/YYYY-MM/` (локально, вне репозитория).
- Создать копию с обезличиванием (убрать ФИО/телеграм/почты).

2) Черновая категоризация
- Темы: Interaction, Economics/ROI, Field day/Onsite, Innovation/NDT, Practice/Examples, Other.
- Тон: Positive, Neutral, Negative.
- Инструмент: ручная пометка в Google Sheets или локальный скрипт (по словарям из `docs/aos/keywords/topics_ru.txt`).

3) Сводка
- Считать частоты по темам и тону; выбрать 3–5 репрезентативных цитат.
- Записать итоги в `docs/aos/reports/YYYY-MM-comments-topics.md`.

4) Интеграция в отчёт
- Ссылка на сводку тем включается в отчёт волны AOS.
- Выводы (3 правки) привязываем к повторяющимся темам.

Примечание: если появится потребность в автоматизации — оформить мини‑скрипт с регулярками/словари. До этого шага — ручная пометка достаточно быстрая.
