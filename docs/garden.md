# Digital Garden

Персональный цифровой сад — коллекция заметок, инсайтов и ссылок в свободной форме.

**Репозиторий:** https://github.com/look85-ops/digital-garden  
**URL:** https://look85-ops.github.io/digital-garden/  
**Платформа:** GitHub Pages

**Статус:** живёт отдельно, не склонирован локально. В этом репозитории — только ссылка.

**Как обновлять:** клонировать `look85-ops/digital-garden`, добавить/изменить заметки, коммит → автодеплой на GH Pages.

**Содержание:** заметки по L&D, AI, методологии, карьерные размышления — формат цифрового сада (связанные заметки, grow-as-you-go).
